Mac配置交换机

发表于 2017-11-17 | 分类于 硬件

简介

主要介绍mac系统用usb-serial线连接control口来配置交换机，并导入导出配置

环境

  • OS版本: 10.10.2
  • 所需线缆：usb-serial转接口
  • 所需软件：minicom，可通过brew安装

Mac连接交换机

  1. 将交换机和电脑通过USBserial线缆连接

  2. 通过minicom -s 对连接方式进行设置

  3. 选择 Serial port setup

  4. 按照A-G字符进行选择所需修改项，最终修改成如下格式：

  5. 设置完毕之后ESC退出即可连接到交换机

服务及网络配置

  1. Mac环境下开启tftp

  2. 在/private/tftp目录下建立权限为666的配置文件

  3. 配置交换机管理IP与tftp的ip相通，并将Mac电脑IP设定成特定值如192.168.1.1/24,且将Mac电脑和交换机端口直连
